Local Build & Installation Guide

This repository contains the complete native Android Jetpack Compose codebase. Follow the commands to compile and sign your custom offline APK package.

1

Setup JDK and SDK Environments

Ensure JDK 17+ and the Android Platform SDK (API 34) are configured on your development terminal.

2

Compile with Kotlin Compiler & Gradle

Run the release gradle task to assemble debug or signed bundles automatically.

3

Transfer and Sideload

Install the compiled package direct onto standard Android devices with full security certificates intact.
